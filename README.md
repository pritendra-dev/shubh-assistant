# Shubh Tech Assistant (PRO) 🚀

यह एक Python-based सिस्टम मॉनिटरिंग टूल है जिसे मैंने अपने **Termux** एनवायरमेंट के लिए बनाया है।

### ✨ खास फीचर्स:
* **Network Scan:** लोकल नेटवर्क पर मौजूद डिवाइसेस को खोजना।
* **Storage Analysis:** आपके होम डायरेक्टरी की टॉप 5 बड़ी फाइलों की लिस्ट।
* **System Info:** IP एड्रेस, इंटरनेट स्टेटस और RAM यूसेज की जानकारी।
* **Automated Logging:** हर एक्टिविटी का रिकॉर्ड `tech_log.txt` में सेव होता है।

### 🛠 कैसे इस्तेमाल करें:
1. इस रिपॉजिटरी को क्लोन करें।
2. `python assistant.py` चलाएं।
